A minimalistic solution to send newsletters from your dashboard, with a subscribers manager, HTML templates and a shortcode to display a subscription opt-in form.

== Description ==

Do you need a simple plugin to send newsletters from your dashboard without complications?

Then you have found the right solution. Banana Newsletters lets you manage subscribers from your dashboard, as well as let users subscribe from the frontend, via a simple shortcode that you can insert wherever you want.

You will be able to send out newsletters to all your subscribers in just a few clicks, with a custom HTML template that you can edit in the plugin settings.

== Installation ==

1. Upload the plugin folder to your /wp-content/plugins/ folder.
1. Go to the **Plugins** page and activate the plugin.

== Frequently Asked Questions ==

= How do I add subscribers? =

You can visit the subscribers section in the admin area and add them there. Or you can also add the [display_subscribe_form] shortcode to display a subscription form.

= How to uninstall the plugin? =

Simply deactivate and delete the plugin.
